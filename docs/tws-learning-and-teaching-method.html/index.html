思沃学院的教学法
我们发现一个现象，一个人学不会编程不是学不会编程，而是缺少很多习惯。我们曾经在面试学生的时候，问过他们一个问题，你有没有一项需要后天锻炼的技能是远胜同龄人的，如果回答是Yes，那么我们在后续的培养中就会发现他的成长速度是很快的。所以我们有一个理论，你学会了一项这种技能最重要的是获得了对于成就一件事很重要的一组习惯。有了这组习惯，你做什么都能做好。
所以我们后来认为我们的重心是培养习惯，只不过刚好通过编程这个技能来培养。那么这些习惯是什么呢？我们认为是：
